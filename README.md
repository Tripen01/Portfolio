# FarmInput: User Policy & Governance Documentation
## Overview
As part of a collaborative team for the **FarmInput App**, I drafted the User Privacy Policy and Terms of Service. My goal was to ensure legal compliance while keeping the language accessible for our primary users—farmers and agricultural stakeholders.

## Contributions
* **Privacy Policy:** Drafted sections on data collection and storage.
* **Accessibility:** Simplified complex legal jargon into plain English.
* **Collaboration:** Used version control to merge my drafts with the dev team's requirements.

## Skills Demonstrated
* Legal-Technical Writing
* Collaborative Authoring
* Plain Language Editing
